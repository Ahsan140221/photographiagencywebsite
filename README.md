# photographiagencywebsite
###### -Ahsan Ali
The website is for a photography agency named Rajaphotographi.

Link to the Website [**Rajaphotographi**](https://ahsan140221.github.io/photographiagencywebsite/)

The images used in this website are stock images, these images could be changed according to the requirements.

## Tools & Technologies Used :
- Microsoft Visual Studio Code
- HTML5
- CSS3
- Javascript
- Bootstrap
- FontAwesome Icons
- jQuery

## Plugins and Libraries used :
- [Animate.css](https://daneden.github.io/animate.css/)
- [Wow.js](https://github.com/matthieua/WOW)
- [Magnific Popup](https://github.com/dimsemenov/Magnific-Popup)
- [Easing Nav](https://jqueryui.com/easing/)
- [Lightbox](https://github.com/lokesh/lightbox2)
## Features of Website
- 100% Responsive
- Cross-Browser Compatible
- Mobile Friendly
- Navigation - Smooth Scroll
- Scrollspy Effect
- FontAwesome Icon Integration
- Modern plugins and libraries used
- Modern CSS Animations
- Responsive Gallery - Lightbox
